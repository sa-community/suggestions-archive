I've looked into this idea before and decided it was too complex / hacky. The thing is, to get the hacked monitors to show you have to create a real variable in the project with the ID set to sensing_mouse_x or something like that. The addon would have to create these variables when the user checked the boxes (so it didn't just spam vairables on every project) than probably delete them when they're not getting used, it would also probably have to hide them from the user, but if they disabled the addon they would become visable again.
{% endraw %}
