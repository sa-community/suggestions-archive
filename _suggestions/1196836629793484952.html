Disables the ability to move in the Code Editor by dragging the background if turned on. Can be toggled in a similar manner to Turbo Mode in Settings, where the Color Mode and Language options are.
Alternatively, rather than toggling this you could instead opt to only move when holding down the right mouse button instead. This would work with everything else considering right-click context menus show only after releasing the mouse button, and left-click drags would already move and execute code blocks.
