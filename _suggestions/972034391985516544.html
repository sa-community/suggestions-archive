Debugger

Adds a new "debugger" window to the editor. Allows for logging into the "Logs" tab of the debugger window using the "log", "warn" and "error" blocks. The "breakpoint" block will pause the project when executed. All running threads can be viewed in the "Threads" tab of the debugger, and when paused the "Step" button can be used to execute the next block. A graph of frames per second and number of clones can be viewed in the "Performance" tab.
