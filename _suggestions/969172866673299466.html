Basically it only shows in the variable tab variables used in the sprite you are currently in...Yes you can technically use another addon to seperate "this sprite only" and "for all sprites" variables, but you cant always use "this sprite only" variables even if they are only used in 1 sprite, e.g. clones. This would dramatically cut down on clutter, especially if you have heaps of variables in a compex project. You have a toggle at the top of the variables list so that you can disable this when you want to add existing variables to new sprites (Otherwise they wouldn't show up) You can also still search for these variables and add them that way. (This would probably be a part of the data category tweaks addon)
