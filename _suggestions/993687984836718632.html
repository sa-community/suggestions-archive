That's cool, though I had to test it to find out it did that (this really should be added to the description of the addon). However, the enhanced clean up blocks+ feature overrides this unadvertised feature of the "Snap scripts to grid" addon. This is probably a change that needs to be made to the "Snap scripts to grid" addon.
