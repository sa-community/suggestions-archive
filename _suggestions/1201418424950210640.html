Once you start TDD you can never turn back...
Add unit testing as an addon! For those of you who don't know, unit testing and test-driven-development is an important practice used in industry. It helps to write cleaner, modular code, catch errors early, and makes sure the code works, every time. Whenever you make a change, re-run all the tests and if they pass, you are good to go! In my experience I have found it actually speeds up writing code, since bug-catching is done quickly and efficiently.
What this addon might look like:
- Identify all* function blocks beginning with test_ (or similar)
- Run each function one after the other
- Test with "assert" blocks - ie. assert_equal(x, y)
- Output errors where x â‰  y
- if error, then stop that function
- else, if end of function, that specific test case has passed
- Show a summary at the end (tests passed, tests failed, total ran)
  • Test a single function, or all within the sprite, or the entire project
This addon fits well into the existing debugger addon, and could make use of the logs interface for output.
