from @goldenfire644 in a dupe suggestion:
The way I see this working is that there would be a simple little button (perhaps an image of the crosshair at the middle of the costume editor) on the costume editor.
When pressed:
If nothing is selected, then it centers everything in the editor as a group.
If part of your costume is selected, then it only centers that part.
I've been working on a project recently that involves a lot of costumes in the middle of the costume editor. It takes so much time to select multiple shapes and drag them to the middle. I find that more often than not, I want my costume centered, and this would be a big time saver
Possible improvements:
A keyboard shortcut alternative, because this would be to decrease the time spent moving your mouse around.
Please please please someone add this. It would be so helpful!
I'm sorry, I know this is a duplicate, but no one has put any tags on it and it is a really good idea that I have found myself needing.
