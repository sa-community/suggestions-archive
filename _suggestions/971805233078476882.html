When making a custom block, you can set it to for all sprites or for only this sprite like variables. Then (if its public) you can use the block in other sprites. When activating it in another sprite (you can use it just like other blocks) the define block gets created in the other sprite too. When collapsing definer addon is activated too, it will be just a little square, that you can expend again with a right click.
