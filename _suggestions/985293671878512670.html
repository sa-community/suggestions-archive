This addon would be a multiple selection for blocks. For example, if you hold shift (or other key) and start clicking blocks, you would be selecting these, then you can do the same action for every selected block, like delete for example, or if these are the same blocks, you can change their value. You could copy and paste blocks too. I know there's the duplicate option and it duplicates the stacks, but with this you could duplicate blocks even if these are not stacked. Hope that someone reads this suggestion and supports it.
