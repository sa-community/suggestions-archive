I would like to be able to choose which blocks go in which category, and possibally even make my own custom categories, with unique colours and different blocks in each.
I was thinking there would be a small settings icon at the bottom of the categories column, where you can add, edit and remove the categories, and then you can drag and drop different blocks into different categories in there. it would also work really well to organise your custom blocks too.
When non-scratch addons user press see inside on a project with customised categories, they will see the original categories. and all custom blocks from any categories will be moved back to the custom blocks category. But if a scratch addon user presses see inside, if they have the customisable categories addon on, they will see the custom categories.
