Conversations on Scratch are slow, which means you end up working on a project while talking to another person. The messages badge count only updates every 5 minutes or whenever you refresh the whole page (whether this be the project editor you're in or the messages page in a separate tab).
Rather than doing this, which is slower and may use bandwidth, you could hover over the message page button for about a second and it will fetch an updated message count from the Scratch Servers. This is most useful when used in conjunction with the Messages in editor addon.
I am referring to the vanilla Scratch messages button, not the Scratch Addons button.
