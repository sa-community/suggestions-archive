Mark a variable/list as "constant" to hide it from blocks that modify the variable/list. Perfect for variables or lists that you want to avoid modifying by accident. If this isn't possible, at least hide capitalized variable/list names, because capitalized names usually suggest that they are meant to be constant. Only show the variable/list name for the first modifier block that uses it, since it will need to be modified to be set to a value in the first place.
