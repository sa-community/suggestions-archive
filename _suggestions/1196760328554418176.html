While working on a project I needed a lot of folders due to having a large number of assets in a sprite.
However, folders add an option to the right-click context menus to have costumes added to them, and if there are a lot of folders it can take up a lot of space.
So, a solution would be to simply add scroll bars to the menus if they have too many options (around 6 or 8 should do). Not only would this fix the folder situation I'm having, it could also be beneficial for future updates that require taking up space in these menus.
