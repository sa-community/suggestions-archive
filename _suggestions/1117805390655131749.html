When you use "store settings in project" on turbowarp, it creates a comment that stores advanced features like 60fps, hq pen, ... to enable when loading the project on turbowarp. This idea suggests that 60fps mode should be enabled when loading the project on scratch if it is specified in the turbowarp configuration.
