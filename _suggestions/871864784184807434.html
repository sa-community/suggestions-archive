Addon to show limits for various things, in the project window or a seperate menu
There may be a little button that you can click when you're worried about your project exceeding a limit. When clicked on the button, a menu with some info will be seen.
To make this more advanced, a suggestion bar that tells what limits your code may exceed in its current form could be added if possible.
Status on this bar will be renewed each time the menu is opened, and there might also be a button to instantly refresh the status without re-opening the menu.
(not too sure about the last one, though, since you need to change some code or let the project run some more to actually see a change, and for that you'd probably click out of the menu.)
Might be better for Turbowarp, because it provides a way to remove some limits or extend them to some point. That way this isn't only a general informative text, but rather a dynamic text showing the current settings.
