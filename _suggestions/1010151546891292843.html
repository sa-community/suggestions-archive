Increase the functionality of duplicating blocks by holding the alt key and dragging the block, by giving "set variable to" blocks special treatment. When holding ctrl in addition to the normal alt to duplicate, it would not duplicate the set variable to block, but instead would add a variable reporter block. This would be helpful when setting a variable and then later on in the script actually using it. It would also mean you do not have to navigate to the variables tab in the block menu, if you have lots of variables, find the one you want, and then drag it in, you could simply hover over a set variable to block, hold ctrl + alt, and act like you are duplicating an alternative version of the block.
