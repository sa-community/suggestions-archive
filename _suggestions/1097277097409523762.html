COSTUME AND SOUND EDITORS
-Tab jumps 1 costume/sound lower (if arrows not pressed)
-Shift + Tab jumps 10 costumes/sounds lower (if arrows not pressed)
-Should wrap from final costume/sound to first
-Arrow keys control direction of navigation
-If 10 costumes/sounds are jumped and it tries to switch to a non-existant costume/sound, then it would switch to the last existant costume. Same would apply backwards.
(Example (Down): Costume 26 doesn't exist, so switch to the last costume. Last costume in this sprite is costume 17, so switch to costume 17. If original costume selected was costume 17, then wrap around)
(Example (Up): Costume -5 doesn't exist, so switch to the first costume. If original costume selected was costume 1, then wrap around)
SCRIPT EDITOR (Tab is good enough, but could use a couple improvements)
-Shift + Tab jumps horizontally across scripts (if arrows not pressed)
-Tab should select the nearest block input to the mouse
-Arrow keys control direction of navigation
