It would be helpful to have an add-on that could assist with creating curves in a vector costume. Usually, when you make a curve with the Reshape tool, Scratch will automatically curve it in a way that can sometimes bulge the curve or make it stick out in some scenarios. It would be great if there was an add-on that could change the way curves are automatically arranged when a curve is created.
Additionally, an option for setting "curved" to default (automatically switch a new point in a vector shape to curved) would be helpful in quite a lot more scenarios.
