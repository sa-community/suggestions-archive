When specific comment is added to script it will hide some varibles and messages from lists and context menus in other scripts.
Eg. When you add comment "varibles: x, y, global varibles: g, h, messages: m, n" it will declare them as script varibles and hide them from other scripts. Very useful when you have a lot of varibles.
