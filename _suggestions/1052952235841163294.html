Add keyboard shortcuts to the project player, for example :
ctrl + enter : run the project
ctrl + shift + enter : turbo mode
ctrl + alt + enter : 60 fps mode
ctrl + space : pause
ctrl + 0 : stop the project
ctrl + T : open with Turbowarp
ctrl + F : full screen
ctrl + M : mute
ctrl + L : put a like
ctrl + shift + L : put in favorite
