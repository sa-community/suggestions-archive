When you make a new variable, enter closes the popup and creates the variable. Scratch Tools has a feature which allows the variables to default to "For this sprite only" but..
1. I usually want "For all sprites" and just want an easier way to create them as "For this sprite only"
2. Scratch Tools is not included in the TurboWarp desktop app, which is what I primarily use.
I propose that holding ctrl while pressing enter causes the variable to be made as a "For this sprite only" variable, this would make it so much easier to create them.
