if you've got a grid of values, eg
1,2,3,4
2,4,6,8
3,6,9,12

with the same number of elements in each row, you store all the elements in one list eg 1,2,3,4,2,4,6,8,3,6,9,12, then to get the item in row r, column c (where r and c are 0-indexed) you get the r * 4 + c + 1th item of the list, replacing 4 with the number of columns - the + 1 is bc scratch lists are 1-indexed
