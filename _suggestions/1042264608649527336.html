Not sure if I thought this through enough but there could be a way for users of ScratchAddons and TW to have unlisted projects. A button called "Get Share Link" in the editor could upload the project file as a ZIP file to assets.scratch.mit.edu/hash.zip.
The addon would also involve scratch.mit.edu/projects/editor having a url parameter that accepts a link to a project file and automatically loads it in the editor; an example being I share a project with my friend and get the link https://scratch.mit.edu/projects/editor?project_url=assets.scratch.mit.edu/0cc175b9c0f1b6a831c399e269772661.zip, my friend opens the link, the editor opens the project from the assets URL and can view it. Obviously I didn't work out all the logistics but the basic idea is clear.
