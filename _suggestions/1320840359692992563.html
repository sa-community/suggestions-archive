Using the scratch api to fetch the info of the project, you can see when it was originally created, when it was uploaded, and when it was last modified. Example: https://api.scratch.mit.edu/users/griffpatch/projects/1107467742
Letting users choose what times they want to see would benefit them because a, they could see when it was last modified (maybe it was created a long time ago but was modified yesterday), and b, they could see how long it took to release (using both when it was created and when it was shared)
This could show up in the project info, or near where it says the date the project was shared on the project page.
