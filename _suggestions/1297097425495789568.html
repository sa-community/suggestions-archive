The Script Menu is an idea I have to significantly speed up the process of creating Scratch projects. It functions similarly to the "backpack" feature, but with key differences that make it more accessible and user-friendly.
How it works:
- The Script Menu would function much like the "Insert Block by Name" add-on, but with an enhanced user experience.
- Instead of just inserting a single block or a small script, the Script Menu would display a labelled menu of all scripts that users have saved, whether from the backpack or stored separately.
- This would allow users to quickly find and insert pre-saved scripts, saving time and making project creation much more efficient.
