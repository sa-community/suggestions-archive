HOW
Very simple, all you need to do is right click and press the "Clear list" or "Reset variable" button and its done! (See picture mock-ups)
WHAT
This could be another setting for the "Data category tweaks" addon, as it's not really that big. Could be combined with some other addons like the "Switch variables between etc" addon or my other variable-related suggestion. The "Clear list" (or "Delete all in list for consistency with scratch") would simply remove all entries to the list and leave it with a length of 0, or with a setting could remove the values in the list entries, but keep the entries themself. The "Reset variables" button would simply set the variable's value to a default value, such as 0. This should be a fully customisable setting though, so you could set it as "" (empty) or any number/string.
WHY
I find often I want to reset a variable or list, and if you have a lot of either, its quite a pain to insert a set variable or delete all of list block and find the exact thing you want to reset in the dropdown, even with search. This addon/new setting would help a lot with this.
