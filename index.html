Welcome!
This site is an archive of suggestions for Scratch Addons taken from the now-locked community Discord server.
More information Close
A few things to note:
  • This site attempts to recreate Discord's UI, but not all elements of it were implemented. Please view suggestions in Discord when possible to get the full context.
  • Not all suggestions have been ported to this site. Only suggestions that were answered as "Good Idea" or "In Development", or those left unanswered, were ported. In other words, only suggestions that still have the capacity to be implemented. Rejected and implemented suggestions can be found by searching the Discord server.
  • Suggestions from both the newer suggestions form channel and the pre-October-2022 bot-managed channel were ported. Form suggestions have the emoji, while legacy suggestions have the emoji.
  • Additionally, duplicates of the suggestions that were ported were not removed.
Pull Requests to modify this site are welcome:
  • To (re-)answer a suggestion. Rejected and implemented suggestions are hidden from suggestion lists, but the actual suggestion page will remain as to not break links.
  • To clarify a suggestion title if the original one is too ambiguous.
  • To add a user's Scratch or GitHub account to their author page.
  • To fix bugs and styling issues.
